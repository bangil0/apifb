Mini Facebook API v0.1

Menghasilkan Cookies

API ini menggunakan cookies untuk login akun facebook anda. Cookies yg di input ke API ini harus di encrypt menggunakan base64 alasanya bukan karena keamanan tetapi agar cookies dapat dikirim menggunakan metode GET. berikut adalah bagaimana menghasilkan cookies API ini:

Metode: POST
Url: {{url}}/generate_cookies
Parameter: user, pass

Contoh penggunaan:

import requests as r
data = {'user':'username kamu', 'pass':'password kamu'}
page = r.post("{{url}}/generate_cookies", data=data).json()

Melihat Informasi Saya

Metode: GET
Url: {{url}}/me?kuki=cookies_ente
Parameter: kuki

Contoh penggunaan:

import requests as r
page = r.get("{{url}}/me?kuki=c2FsaXNnYW5zc2FuZ2FkZGQ=").json()

Melihat Liker Postingan

Metode: GET
Url: {{url}}/post/id_post/likes?kuki=cookies_ente
Parameter: kuki

Contoh penggunaan:

import requests as r
page = r.get("{{url}}/post/id_post/likes?kuki=c2FsaXNnYW5zc2FuZ2FkZGQ=").json()

Jika mengeksekusi perintah diatas secara default sistem akan memilih liker bertype suka. Tambahkan parameter "type" untuk melihat liker yang bertype react, untuk valuenya anda bisa lihat dibawah.
      - mantapmantap = love
      - sadna = sad
      - ngamuk = angry
      - ngakak = haha
      - keren = wow

Contoh penggunaan:

Metode: GET
Url: {{url}}/post/id_post/likes?kuki=cookies_ente&type=type_na
Parameter: kuki, type

import requests as r
page = r.get("{{url}}/post/id_post/likes?kuki=c2FsaXNnYW5zc2FuZ2FkZGQ=&type=mantapmantap").json()

Menyukai Postingan

Metode: POST
Url: {{url}}/post/id_post/likes
Parameter: kuki

Contoh penggunaan:

import requests as r
data = {'kuki:'c2FsaXNnYW5zc2FuZ2FkZGQ='}
page = r.post("{{url}}/post/166694224710808&/likes", data=data).json()

Atau

Metode: GET
Url: {{url}}/post/id_post/likes?kuki=c2FsaXNnYW5zc2FuZ2FkZGQ=&method=post
Parameter: kuki, method

Contoh penggunaan:

import requests as r
page = r.post("{{url}}/post/166694224710808&/likes?kuki=c2FsaXNnYW5zc2FuZ2FkZGQ=&method=post").json()
